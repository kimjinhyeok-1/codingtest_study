# [조이스틱]

### 1. 접근 방식
- 0번째 인덱스부터 모든 인덱스를 돌면서 그리디로 풀게됨

### 2. 핵심 로직
- 각 인덱스 값에서 어떤 값들을 비교하는지가 중요 (미리 유턴해서 올까, 갔다가 유턴할까, 그냥 쭉 갈까)

### 3. 시간 복잡도
- for문을 한번만 돌기 때문에 O(N) 이라고 생각

### 4. 느낀 점 / 어려웠던 점
- 솔직히 혼자 못풀었음. 유턴한다는 개념이 안잡혀 있었는데 2번 유턴도 가능하지 않을까? 라는 생각이 지배적이였던 것 같음
- 2번 유턴이 말이 안된다는걸 알고 나서도 for 문으로 i 를 모두 훑으면서 확인하는 법을 알게 됨

### 5. 풀이 시간
- 풀이 보고 풀었음
