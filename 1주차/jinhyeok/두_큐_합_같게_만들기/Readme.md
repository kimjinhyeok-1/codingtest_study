# [두_큐_합_같게_만들기]

### 1. 접근 방식
- 두 개 큐를 이어붙혀서 풀었음

### 2. 핵심 로직
- left, right 나눠서 이분탐색으로 풀었는데
- 따로 queue 두개 해서 관리하는게 정석 풀이라고 하네?

### 3. 시간 복잡도
- for문을 한번만 돌기 때문에 O(N) 이라고 생각

### 4. 느낀 점 / 어려웠던 점
- 적어도 3 * n 을 돌려야 된다고 하는데 아직 완벽하게 이해를 하지는 못함ㅎ
- 그냥 문제 적힌대로 왼쪽에서 꺼내서 넣고 이런거 반복하는거도 괜찮다고 생각함 정석풀이라고 하고
